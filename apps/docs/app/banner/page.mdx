## Banner Props

Props to can be used with the `Banner` component.

| **Name**          | **Type**                                                 | **Default** | **Description**                                                                                       |
| ----------------- | -------------------------------------------------------- | ----------- | ----------------------------------------------------------------------------------------------------- |
| `type`            | `'gray' \| 'info' \| 'warning' \| 'success' \| 'danger'` | `info`      | Determines the Banner component's color variation                                                     |
| `title`           | `React.ReactNode`                                        | `''`        | shows the title of the Banner                                                                         |
| `description`     | `React.ReactNode`                                        | `''`        | shows the subtext description of the Banner                                                           |
| `showCloseButton` | `boolean`                                                | `false`     | shows the close button                                                                                |
| `open`            | `bolean`                                                 | `true`      | shows or hide the entire component                                                                    |
| `showIcon`        | `bolean`                                                 | `true`      | shows or hide top left icon                                                                           |
| `bottomNode`      | `React.ReactNode`                                        | `null`      | ReactNode use to display components under the description                                             |
| `sideNode`        | `React.ReactNode`                                        | `null`      | ReactNode use to display components beside the text block. This will become a column layout on mobile |
