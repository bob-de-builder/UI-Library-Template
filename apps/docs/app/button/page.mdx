## Button Props

The `Button` component extends `React.ButtonHTMLAttributes<HTMLButtonElement>` as well as additional props for customization.

| **Name**    | **Type**                                               | **Description**                                                                              | **Default**           |
| ----------- | ------------------------------------------------------ | -------------------------------------------------------------------------------------------- | --------------------- |
| `intent`    | `'primary' \| 'secondary' \| 'outline' \| 'danger'`   | intent style of the button.                                                                 | `primary`             |
| `size`      | `'sm' \| 'md' \| 'lg'`                                 | size of the button.                                                                          | `'md'`                |
| `asChild`   | `boolean`                                              | If true, the button will render as a different component, specified in the `Slot` component. | `false`               |
| `isLoading` | `boolean`                                              | If true, displays a loading spinner inside the button.                                       | `false`               |
| `leftNode`  | `React.ReactNode`                                      | Element(s) to be displayed before the button content.                                        | `-`                   |
| `rightNode` | `React.ReactNode`                                      | Element(s) to be displayed after the button content.                                         | `-`                   |
| `className` | `string`                                               | Additional classnames to be applied to the button.                                           | `-`                   |
| `disabled`  | `boolean`                                              | If true, disables the button.                                                                | `false`               |
| `children`  | `React.ReactNode`                                      | The content of the button.                                                                   | `-`                   |
